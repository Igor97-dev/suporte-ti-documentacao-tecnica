# 🪟 Estudos de Windows para Suporte Técnico 

Este arquivo contém anotações sobre o sistema operacional Windows, com foco em atividades comuns de **Suporte Técnico / Help Desk / Suporte de TI**.

O conteúdo será atualizado conforme avanço nos estudos e na prática.

---

## 📚 Conceitos Básicos do Windows

- O Windows é um sistema operacional desenvolvido pela Microsoft
- Utilizado amplamente em ambientes corporativos
- Possui interface gráfica (GUI) e ferramentas administrativas
- Muito usado em estações de trabalho (desktops e notebooks)

---

## 🧑‍💻 Funções do Suporte Técnico no Windows

- Auxiliar usuários com problemas no sistema
- Instalar e configurar programas
- Resolver erros comuns
- Criar e gerenciar usuários
- Verificar desempenho do computador
- Prestar suporte remoto
