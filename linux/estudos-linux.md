## 📚 Conceitos Iniciais

- O Linux é um sistema operacional baseado em Unix
- É muito utilizado em servidores
- Possui foco em estabilidade, segurança e controle
- Grande parte da administração é feita via terminal

---

## 📁 Navegação entre Diretórios

Comandos utilizados para se locomover entre pastas no sistema:

- `pwd` — mostra o diretório atual
- `ls` — lista arquivos e diretórios
- `cd` — entra em um diretório
- `cd ..` — volta um nível
- `cd /` — vai para o diretório raiz

---

## 🗂️ Manipulação de Arquivos e Diretórios

Comandos para criar, mover, copiar ou apagar arquivos e pastas:

- `mkdir` — cria diretórios
- `touch` — cria arquivos vazios
- `cp` — copia arquivos ou diretórios
- `mv` — move ou renomeia arquivos
- `rm` — remove arquivos
- `rm -r` — remove diretórios
